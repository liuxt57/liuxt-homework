使用二分法：

```
 Binary_Search(first, last)
    IF (first > last) THEN
        RETURN false
    ELSE 
        set middle to (first + middle) / 2
        set result to item.compareto(middle)
        IF (result = 0) THEN 
            RETURN true
        ELSE 
            IF (result < 0) THEN 
                Binary_Search(first, middle)
            ELSE
                Binary_Sreach(middle, last)

```
